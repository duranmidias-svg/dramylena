# Landing Page - Dra. Mylena Simões

Arquivos prontos para subir no GitHub e publicar na Vercel.

## Estrutura

- `index.html`: página final com CSS, JS e imagens embutidos no próprio arquivo.

As imagens foram convertidas para Base64 dentro do HTML. A pasta `assets/` não é necessária para publicar esta versão.

## Como publicar na Vercel

1. Crie um repositório no GitHub.
2. Envie o `index.html` para a raiz do repositório.
3. Na Vercel, clique em `Add New Project`.
4. Importe o repositório.
5. Em `Framework Preset`, use `Other`.
6. Não precisa configurar build command.
7. Publique.

O arquivo `index.html` precisa ficar na raiz do repositório para a Vercel abrir a página corretamente.
